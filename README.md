# JavaScript Development Internship Projects

## 📌 Project Overview

This repository contains four JavaScript projects completed as part of my Web Development Internship.

The projects demonstrate practical JavaScript skills including DOM manipulation, API integration, array methods, searching, filtering, sorting, and responsive user interface design.

---

## 🚀 Projects

### ✅ Task 1 – Interactive To-Do List

**Features**
- Add new tasks
- Mark tasks as completed
- Delete tasks
- Save tasks using Local Storage

**Technologies Used**
- HTML5
- CSS3
- JavaScript (DOM & Local Storage)

---

### ✅ Task 2 – Weather App

**Features**
- Search weather by city
- Display temperature
- Display weather condition
- Display humidity
- Display wind speed
- Error message for invalid city

**Technologies Used**
- HTML5
- CSS3
- JavaScript
- OpenWeather API

---

### ✅ Task 3 – Random User Directory

**Features**
- Display user photo
- Display full name
- Display email
- Display country
- Refresh button to load a new user

**Technologies Used**
- HTML5
- CSS3
- JavaScript
- Random User API

---

### ✅ Task 4 – Dynamic Product Catalog

**Features**
- Display product cards
- Search products
- Filter by category
- Sort by price (Low to High / High to Low)

**Technologies Used**
- HTML5
- CSS3
- JavaScript

---

## 🛠 Technologies Used
